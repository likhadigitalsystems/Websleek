# Websleek IT Solutions — Website

React + Vite + Tailwind marketing site for Websleek IT Solutions (Philippines).

## Run locally

```bash
cd website
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build for production

```bash
npm run build
npm run preview
```

## Pages

- **Home** — Positioning, differentiators, product offers, qualification
- **Solutions** — OpsCore™, GovFlow™, hosting bundles, retainers
- **Industries** — SME verticals (retail, healthcare, services)
- **Government** — Capability layer for procurement evaluators
- **Case Studies** — Proof of deployment
- **Contact** — System assessment request form

## Brand assets

Logo and favicon: `public/assets/logo.png` (from `Transparent.png` in the parent folder).
